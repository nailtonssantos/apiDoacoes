Quem Somos...

Sistema de Doações é grupo fictício criado como estudo de caso para o curso "Desenvolvimento de um Sistema de arrecadação de alimentos, roupas, e demais itens, utilizando Spring Boot, Angular e Bootstrap 5" disponibilizado gratuitamente no meu GitHub.

O código-fonte pode ser obtido via GitHub, conforme orientações apresentadas no arquivo Readme. Este estudo de caso pode ser utilizado livremente como base para outras entidades/instituições, sem custos para o usuário.
